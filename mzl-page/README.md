# MZL 个人页面

我的个人展示页面，包含摄影、学习、游戏、娱乐等模块。

## 页面结构

- **首页** - 四个模块入口
- **摄影** - 摄影作品展示（6列网格布局）
- **学习** - 学习笔记分享
- **游戏** - 游戏内容分享
- **娱乐** - 娱乐内容分享
- **call mzl** - 联系方式
- **详情页** - 各模块详细内容

## 本地预览

1. 打开 VS Code
2. 安装 Live Server 扩展
3. 右键 index.html -> Open with Live Server

## 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入仓库 Settings -> Pages
3. Source 选择 `main` branch
4. 保存后等待部署

访问地址：`https://MziL-1.github.io/mzl-SelfPage/`
